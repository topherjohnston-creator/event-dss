# event-dss
Real-time weather decision support system for large events using experimental  REFS (Regional Ensemble Forecast System) ensemble forecast data from AWS
Features 60-hour probabilistic forecasts with:
- Risk cards (60-hr max/min by hazard)
- 48-hour timeline (20 × 3-hr blocks) 
- NWS Watches/Warnings/Advisories
- Airport Weather Warnings (AWW)
- Real-time ASOS observations
- Config-driven locations and hazards

Automated updates: REFS every 6 hours, alerts/obs every minute.

⚠️ EXPERIMENTAL, NON-OPERATIONAL, RESEARCH PURPOSES ONLY
